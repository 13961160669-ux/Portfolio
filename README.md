# Research Assistant Portfolio: Reproducible Data Workflows
**Author:** Yeshuang Wu (吴烨霜)

Welcome to my portfolio! This repository demonstrates my ability to build reliable, reproducible, and audit-friendly data workflows for day-to-day research tasks. 

##  Repository Structure

This repository is divided into three concrete, deliverable-oriented projects:

- **`01_data_cleaning/`**
    Scripts for turning raw data (e.g., instrument exports, time-series data) into analysis-ready formats. Features include data caching, missing/outlier handling, and generating traceable processing logs.

- **`02_plotting/`**
    Reusable templates for generating paper/meeting-ready figures (e.g., time-dependent curves, kinetics fitting). Ensures consistent visual styles and includes basic fit diagnostics.

- **`03_gru_replication/`**
    A baseline replication of a sequence model (GRU). Focuses on strict zero-leakage data splitting, experiment tracking (saving configs/logs/checkpoints), and robust evaluation.
